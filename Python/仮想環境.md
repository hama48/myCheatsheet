# 仮想環境の作成
    ```powershell
    > py -m venv .venv
    ```

# 仮想環境の起動
    ```powershell
    > .\.venv\Scripts\Activate.ps1
    ```

# pythonインタープリタを仮想環境に設定
    仮想環境上にインストールしたライブラリを認識させるため

    1. `Ctrl + Shift + P`
    2. `Python: Select Interpreter`
    3. `Enter interpreter path...`
    4. ./.env/Scripts.exe を選択
    5. VSCodeを再起動

# 仮想環境の停止
    ```powershell
    (.venv) > deactivate
    ```

# 仮想環境の削除
    ```powershell
    > rm ./.venv
    ```
